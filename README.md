# ¿Qué sucede si Porta Hnos explota?

**Se realizó un estudio de carga de fuego a partir de los datos públicos de la empresa y los resultados son preocupantes. Aquí una visualización del radio explosivo a partir del informe y los productos desarrollados por la empresa**
